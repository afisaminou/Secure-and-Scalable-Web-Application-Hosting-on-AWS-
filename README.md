# Secure-and-Scalable-Web-Application-Hosting-on-AWS-
This architecture supports a secure, scalable web application hosted on AWS. It spans two Availability Zones for high availability, with public subnets hosting an Application Load Balancer and NAT Gateways, private subnets for EC2 Auto Scaling and RDS Multi-AZ, and integrated S3 and CloudWatch for asset delivery and monitoring.
